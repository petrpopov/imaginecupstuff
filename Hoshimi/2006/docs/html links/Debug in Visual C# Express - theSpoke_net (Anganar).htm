the express version can debug a dll but you have to edit a file by hand, the file is <project>.csproj.user , just put the complete path to PHServer.exe between <StartProgram></StartProgram> and you can start debugging via F5.
Here I suppose your dll is generated in the right place for the server to find it.

From my short usage of the express version it really looks like it can do nearly everything the complete version can do, some things works in the different way and others are not configurable in the ide but even. I think some things are missing in the interface editor too.
