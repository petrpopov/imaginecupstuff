 Flip wrote:
Ok, thanks for this point.
And for the reference, is it still valid (null or not ?) if the ennemi bot is removed from the collection ?

If i understand what you wonder, you want to use a NanoBotInfo you stored some turns ago?

This is not useful, because the NanoBotInfo that you get in OtherNanoBotinfos is like a snapshot of a bot at a turn, if the bot is moving, the NanoBotInfo who refers to it won't be updated, but a new NanoBotInfo will be added in OtherNanoBotInfos.

Hope this helps.
