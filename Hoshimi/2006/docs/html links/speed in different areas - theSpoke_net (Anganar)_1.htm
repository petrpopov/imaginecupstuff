You absolutely go faster. The answer is: there is a modifier when going against the stream (+2) on the difficulty to traverse the node. When you're in the stream and not going against it, the modifier is -2. I don't know if the perpendicular issue is a bug or not, but I think it isn't. Practically speaking, that means that you'll traverse it 2 turns faster... with no negative times, of course. So, if you're in a vessel (normal cost 1) going with the stream (or "in" the stream), it'll take you 0 turns to traverse it, and if you're against the stream it'll take you 3 turns to traverse it. I don't understand why everybody's being so elusive with this question!!!!!!!!!!!
