BEWARE:

There is AreaType (which set the speed cost of the cell) AND Region.

Region which can be Red, White or Blue is only for the aspect in the 3DViewer

AreaType which can be Red (lowDensity), Green (Medium) and Blue (High) is the cost for speed.

So take only AreaType in consideration in your code
