Hey Juliano,

To answer your questions,

-Yes, this is fixed, so you don't need to do anything about it. But you should, as laurens mentioned. Like exchange them with what richardc said, Tissue.XXX. If you cannot btw. find Tissue in the MyNanobots.cs, use the given references like _player(.Tissue).

-Yes you can issue orders like move to those points (or anywhere else), basic pathfinding is implemented. But you should improve it.

-Choosing the number of nanobots of each type you will use is part of your task. The total amount should always stay near maximum, except you got some really strange tactics which are so time-critic for your NanoAI that it cannot build the maximum number.

Have fun
