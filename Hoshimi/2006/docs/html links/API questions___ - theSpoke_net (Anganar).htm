Who's Pierre?
I have no idea, I suspect it's someone from the Hoshimi tale thing they made up on the website but I never read it. Else it's some common term for 'the enemy'.. who cares?

To your second question, what they mean is if you have:
D C D
C P C
D C D

as an array of pixels, and your bot is P, then your bot can move directly to the C spots but not directly to the D spots, it would have to get to a D spot in two moves. Remember that the 3d map is only a cosmetic byproduct generated off a 2d pixel array aka a simple bitmap.
You can pass any (reachable) point on the map to the function u want, the prewritten pathfinding algorithm will figure out a way to get there.
